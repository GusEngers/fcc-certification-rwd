# DISEÑO WEB RESPONSIVO

Colección con los proyectos para obtener la "Certificación de Diseño Web Responsivo" de freeCodeCamp.org.

- **Página de Homenaje**
- **Página de Documentación Técnica**
- **Landing Page**
- **Página Web de Portafolio**
